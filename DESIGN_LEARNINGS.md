# LUEUR Design Learnings

## 目的

trendペルソナは、商品情報より先に「自分の気分に合うか」「SNSで見つけた時の高揚感が続くか」を判断する。LUEURでは、K-beautyの透明感、Y2Kのきらめき、ガーリー編集誌の甘さを組み合わせ、発見感と欲しい気持ちを作る設計にした。

このブランドは架空ブランドであり、実在の店舗・商品ではない。

## 設計したこと

- ファーストビューで淡いラベンダー、桜ピンク、シャンパンゴールドを見せ、やわらかい自己表現の方向性を伝える。
- ヒーロー写真は「自然の雰囲気」ではなく、ファッションキャンペーンとして読める素材へ変更し、`mirror / ribbon / gloss` のラベルを重ねた。
- 商品グリッドは完全整列させず、雑誌のページをめくるような発見感を出した。
- CTAは丸く、薄い白背景とゴールド線で、強い購入圧ではなく「見に行く」軽さを作った。

## CSS

主要なCSS判断:

```css
:root {
  --bg: #FAF7FF;
  --accent: #E8C4D4;
  --accent-gold: #C9A84C;
  --accent-deep: #7B5EA7;
  --font-display: 'Cormorant Garamond', 'Noto Serif JP', serif;
  --font-body: 'Nunito', 'Noto Sans JP', sans-serif;
}

.hero-photo {
  border-radius: 34% 34% 6px 6px;
  box-shadow: 0 32px 70px rgba(123,94,167,.16);
}

.arrivals-grid {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
}
```

CSSは柔らかい色面、丸み、薄い透過、軽い影で構成した。輪郭を強くしすぎるとstreetやminimalに近づくため、境界は淡く扱っている。

## アニメーション

- ローダーは1.4秒で消し、SNS遷移後の短い期待感を作る。
- 商品カードはIntersectionObserverで順番にフェードインする。
- 商品ホバーは `scale(1.06)` とピンクのオーバーレイだけに抑え、かわいさと操作感を両立した。
- マーキーはブランド語彙を流し、編集誌のリズムを作る。

## フォント

- Display: `Cormorant Garamond`
- Body: `Nunito`

セリフ体で甘さと高級感を作り、丸いサンセリフでモバイル本文の読みやすさを担保した。日本語はNoto系へフォールバックする。

## ボタン

丸いピル型、ゴールドの1px線、薄い白背景を使った。強い黒ボタンにしないことで、購入圧を下げ、閲覧体験を軽くしている。

## UI/UX

- ナビは中央ロゴでブランド名を常に記憶させる。
- モバイルでは商品グリッドを2カラムにして、SNS後の短時間ブラウズに合わせる。
- フッターにはニュースレターUIだけを置き、決済やログインのような実機能は持たせていない。

## レイアウト

非対称グリッドと大きな余白を組み合わせた。整いすぎるとFORMに近づくため、あえてカードの高さと位置に揺らぎを作っている。

# LUEUR — design-apparel-trend

> **"Glow in your own light"**

A design study exploring a fictional youth apparel brand tailored to the **trend persona** — SNS-driven, K-beauty-influenced, Y2K aesthetic, targeting Japanese women aged 20–23.

LUEUR is a fictional brand created for this design study. It is not a real brand, store, or product.

## Overview

| | |
|---|---|
| **Brand** | LUEUR |
| **Persona** | trend |
| **Live Site** | [design.apparel-trend.riumu.net](http://design.apparel-trend.riumu.net/) |
| **Custom Domain** | `design.apparel-trend.riumu.net` *(HTTPS certificate pending)* |

## Design Concept

- **Color**: Lavender mist × cherry blossom pink × champagne gold
- **Typography**: Cormorant Garamond (display) × Nunito (body)
- **Aesthetic**: Parisian petit brand meets K-beauty editorial magazine
- **UX**: Soft easing, film-grain hero, ribbon-style product tags, immersive scroll

## Tech Stack

- Pure HTML + CSS Custom Properties + Vanilla JS
- Google Fonts CDN
- No framework, no build step — GitHub Pages ready
